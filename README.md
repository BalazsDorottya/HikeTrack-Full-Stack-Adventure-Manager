# HikeTrack 🥾

A full-stack web application for outdoor enthusiasts to discover trails, track personal achievements, and manage their hiking history. 

## 🏔️ Project Overview

HikeTrack bridges the gap between a simple trail directory and a personal fitness log. Built with a **Java** backend and a **TypeScript** frontend, this platform allows users to build a personal profile, "favorite" upcoming adventures, and maintain a tally of how many times they have conquered a specific peak.

### Key Technical Highlights:
* **Type-Safe Frontend:** Built with TypeScript to ensure UI stability and a seamless developer experience.
* **Robust Backend:** A Java-based REST API handling complex filtering and user data persistence.
* **Relational Tracking:** Implements logic for users to log multiple completions of the same hike, moving beyond simple "check-off" lists.
